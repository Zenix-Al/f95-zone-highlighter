### Requires F95Zone Ultimate Enhancer (core)

Install and enable the [F95Zone Ultimate Enhancer core script](https://greasyfork.org/scripts/546518-f95zone-ultimate-enhancer) first. Site Repair is a core-managed add-on and will not run without it.

### F95UE Site Repair Add-on

Formerly published as **F95UE Image Repair Add-on**.

Provides independently controlled repairs for F95Zone:

- Retries broken `attachments.f95zone.to` images on thread pages.
- Allows the image retry limit and retry interval to be configured from the core add-on panel.
- Recovers eligible Latest Ajax failures with one bounded retry.
- Can hide repair-activity toasts and the active image-repair status indicator.
- Cleans up its styles, observers, retries, and page patches when disabled.

Enable Site Repair and its individual repairs from the core add-on panel.
