# Directory Map

Top-level `src/ui` layout (high level):

- assets/: css and `ui.html`
- components/: UI components and subfolders (`addons`, `tag-search`, `featureHealth`)
- helpers/: `cssInjector.js`, `updateColorStyle.js`
- renderers/: `renderSetting.js`, `createInput.js`, `settingsSection.js`, etc.
- settings/: `globalSettings.js`, `colorSettings.js`, `tagsSettings.js`, `modalLifecycle.js`, `panelNavigation.js`
- settingsRuntime/: prefs and sections registry
- getShadowRoot.js, index.js

Use this file to locate focused docs for each subsystem.
