# Known Analysis Limitations

This set of documents was created from a static source review. Not performed here:

- building the userscript
- running automated tests
- exercising the UI in a browser
- verifying every selector in `assets/document.css`
- confirming runtime sanitization of add-on-provided markup
- measuring performance with many tags or add-ons

Items marked Verify/Potential/Risk are review targets, not confirmed defects.
