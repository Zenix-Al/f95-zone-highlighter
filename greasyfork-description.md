### F95Zone Ultimate Enhancer

**The all-in-one beast for F95Zone, now with official add-on support.**

Upgrade your F95Zone experience with thread highlighting, custom tags & colors, wide layout, auto-refresh, diagnostics, and much more.

The core script now provides a solid foundation + **modular add-on system**. Some specialized or heavier features have been moved to optional add-ons so the core stays lean and fast.

### Core Features

**Global**

- Notification dismissal
- Powerful tag management (search + preferred/excluded/marked lists with drag-to-reorder)
- Full color customization with sleek dark picker
- Optional cross-tab settings sync
- Feature health diagnostics

**Latest Updates page**

- Auto-refresh + web notifications
- Wide layout + dense grid option
- Latest overlay with intelligent scoring
- Hover tag coloring using your preferred/excluded rules

**Thread page**

- Wide thread layout
- Collapsible signatures
- Thread tag overlay (preferred / excluded / marked)

### Official Add-ons

You can enable or disable these individually from the main config panel:

- **[Image Repair Add-on](https://greasyfork.org/en/scripts/572502-f95ue-image-repair-add-on)**  
  Automatically retries broken attachment images (`attachments.f95zone.to`)

- **[Library Add-on](https://greasyfork.org/en/scripts/572506-f95ue-library-add-on)**  
  Save threads to your personal library with notes, ratings, status tracking, advanced search, pinning, and full import/export

- **[Masked + Direct Download Add-on](https://greasyfork.org/en/scripts/572503-f95ue-masked-direct-download-add-on)**  
  "Resolve" buttons on masked links + smart automation for Buzzheavier, Gofile, Pixeldrain, and Datanodes

- **[Latest Filters Add-on](https://greasyfork.org/en/scripts/573076-f95ue-latest-filters-add-on)**  
  Adds saved filters on the Latest Updates page with searchable presets and active filter tracking

- **[Halloween Theme Add-on](https://greasyfork.org/en/scripts/573070-f95ue-halloween-theme-add-on)**  
  Applies the Halloween logo/background theme with simple apply and teardown lifecycle

- **[Example Add-on](https://greasyfork.org/en/scripts/583616-f95ue-example-add-on)**  
  Developer playground for the current add-on API, useful for testing core bridge actions and extension patterns
